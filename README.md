Instructions:
Preparation: Open Steam first.

Launch Game: Launch the game via this tool (This disables EAC and forces Offline Mode).

Equip Junk: In-game, equip a set of "Junk Relics" (unwanted trash items) to target for modification.

Load Data: Click [Get Current Relic Info] or select a [Loadout Template] from the menu.

Modify: Adjust stats as needed, then click [One-Click Modify].

Refresh: To verify changes, swap your equipment set or restart the game.
